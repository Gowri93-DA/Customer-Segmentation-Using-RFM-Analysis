# Customer-Segmentation-Using-RFM-Analysis
This project applies RFM (Recency, Frequency, Monetary) analysis to segment customers based on purchasing behavior and identify high-value and at-risk segments.  The goal is to transform raw transaction data into actionable customer insights.
